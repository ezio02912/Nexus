# Mobile App

Placeholder cho Flutter mobile app.

Chưa tạo source Flutter ở phase này để tránh khóa kiến trúc app quá sớm. Khi bắt đầu triển khai, tạo app Flutter trong folder này và giữ các layer đề xuất:

- `lib/app`
- `lib/core`
- `lib/features`
- `lib/shared`
- `test`

Mobile sẽ gọi API qua Gateway/BFF, không gọi trực tiếp từng service.
