# Development Roadmap

## Phase 0: Foundation

- Monorepo.
- API Gateway.
- PostgreSQL, Redis, RabbitMQ, MinIO.
- Shared building blocks.
- Health checks.
- Docker Compose.

## Phase 1: Core Services

- Identity.
- Tenant.
- Permission.
- Audit.
- File.
- Notification.
- Workflow.
- Numbering.

## Phase 2: CRM + Sales

- Customer, contact, lead, opportunity.
- Quotation approval.
- Contract.
- Sales order.

## Phase 3: Inventory + Purchase

- Product, warehouse, stock in/out.
- Purchase request/order.
- Goods receipt.

## Phase 4: Invoice + Accounting Core

- Invoice lifecycle.
- Payment.
- Receivable/payable.
- Chart of accounts.
- Journal entry.

## Phase 5: HRM + Attendance + Payroll

- Employee profile.
- Attendance and leave.
- Payroll calculation and approval.
- Payroll to accounting event flow.

## Phase 6: Reports + Audit Trace

- Ledger reports.
- Receivable/payable reports.
- Inventory reports.
- Payroll reports.
- Source document trace.
